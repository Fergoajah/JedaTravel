### Cara Instalasi

```bash
git clone https://github.com/Fergoajah/JedaTravel.git
cd JedaTravel
flutter pub get
flutter run
```

## Build Aplikasi
Untuk membangun APK versi release:

```bash
flutter build apk --release
```

File APK akan berada di:
build/app/outputs/flutter-apk/app-release.apk
